Flashcards
¿Qué es un section?
La etiqueta section es una etiqueta de tipo bloque que permite contener elementos de un código genérico. Se diferencia del div, en que esta etiqueta encierra una parte del código, pero no necesariamente debe tener una intención o semántica.
¿Qué es un diseño responsivo?
Un diseño web responsivo es aquel que permite escalar el mismo diseño a distintos tamaños para todos los dispositivos. Esto comenzó gracias a la necesidad de escalar diseños de páginas web a dispositivos celulares y garantizar un uso cómodo en todos los dispositivos
¿Qué es la etiqueta UL?
Las unordered lists (UL), como su nombre lo indica, son etiquetas que contienen listas no ordenadas
¿Por qué es importante tener un archivo css?
Un css permite estilizar toda el diseño de la página web, cambiandole los atributos por defecto a las etiquetas, de esta manera se permite tener una página más estilizada y acorde a la modernidad.
¿Qué es un Span?
Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea.
Java
Java es un lenguaje de programación y una plataforma informática que fue comercializada por primera vez en 1995 por Sun Microsystems. Hay muchas aplicaciones y sitios web que no funcionarán, probablemente, a menos que tengan Java instalado, y cada día se crean más. Java es rápido, seguro y fiable.
¿Para qué sirve la etiqueta form?
Representa una sección de un documento que contiene controles interactivos que permiten a un usuario enviar información a un servidor web.
¿Que es html?
El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.
¿Qué es una pagina web?
Una página web, página electrónica, página digital o ciberpágina, ​​ es un documento o información electrónica capaz de contener texto, sonido, vídeo, programas, enlaces, imágenes, hipervínculos y muchas otras cosas, adaptada para la llamada World Wide Web, y que puede ser accedida mediante un navegador web.
¿Qué es Internet?
Internet es una red de computadoras interconectadas a nivel mundial en forma de tela de araña. Consiste de servidores (o "nodos") que proveen información a aproximadamente 100 millones de personas que están conectadas entre ellas a través de las redes de telefonía y cable.
¿Qué es un servidor?
Un servidor es un sistema que proporciona recursos, datos, servicios o programas a otros ordenadores, conocidos como clientes, a través de una red. En teoría, se consideran servidores aquellos ordenadores que comparten recursos con máquinas cliente.
¿Qué es un index?
El archivo index.html es la página principal del dominio al cual se accede desde Internet, lo que vemos en primer lugar a la hora de acceder a ese nombre de dominio.
¿Qué es el body?
Esta etiqueta contiene todo el contenido de nuestra página web. Acá irán las secciones, los divs, las clases, entre otros.
¿Para que se usa el término class?
Class sirve para identificar elementos de un html con un nombre específico, el cual podemos llamar desde un archivo css.
¿Qué es la etiqueta META?
META de "metainformation" - metainformación. Sirve para aportar información sobre el documento. De esta manera, nuestro navegador toma en cuenta los atributos dentro de la etiqueta META para mostrar al usuario, como pueden ser los enlaces a los archivos css o el nombre de la página web, el icono, entre otros.
¿Qué es la etiqueta article?
Representa una composición auto-contenida en un documento, página, una aplicación o en el sitio, que se destina a distribuir de forma independiente o reutilizable, por ejemplo, en la indicación. Podría ser un mensaje en un foro, un artículo de una revista o un periódico, una entrada de blog, un comentario de un usuario, un widget interactivo o gadget, o cualquier otro elemento independiente del contenido.
IMG
La etiqueta img es un elemento html que sirve para mostrar imagenes
¿Qué es el grid layout?
CSS Grid layout contiene funciones de diseño dirigidas a los desarrolladores de aplicaciones web. El CSS grid se puede utilizar para lograr muchos diseños diferentes. También se destaca por permitir dividir una página en áreas o regiones principales, por definir la relación en términos de tamaño, posición y capas entre partes de un control construido a partir de primitivas HTML.
List
Como su nombre lo indica, son etiquetas para crear una lista que tienen como elementos las etiquetas ul que serán los contenidos de esa lista.
¿Qué es una base de datos?
Una base de datos es un conjunto de datos pertenecientes a un mismo contexto y almacenados sistemáticamente para su posterior uso. Dentro de las páginas web son muy usadas, ya que normalmente permiten obtener la información necesaria para el correcto funcionamiento de la misma.
